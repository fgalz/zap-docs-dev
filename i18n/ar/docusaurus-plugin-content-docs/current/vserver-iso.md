## تركيب ملف ISO
لتركيب ملف ISO، انتقل إلى لوحة إدارة VPS الخاصة بك وافتح **الإعدادات → ملفات ISO**. اختر ملف ISO الذي تريده، واضبط ترتيب الإقلاع، ثم أعد تشغيل الخادم.

![img](https://screensaver01.zap-hosting.com/index.php/s/tszMKbqDSa3AaLy/download)

بعد إعادة التشغيل، يمكنك الاتصال بخادمك باستخدام عميل VNC وبدء تثبيت وتكوين نظام التشغيل.

![img](https://screensaver01.zap-hosting.com/index.php/s/q6WoDMq8pxn72oG/download)

:::warning حالة الاتصال أو الاتصالات البعيدة بعد تركيب ISO

بعد تركيب ملف ISO، لم يتم إعداد الخادم بعد. نتيجة لذلك، لن يتم عرض **حالة الاتصال** في اللوحة، ولن تتوفر **اتصالات RDP** أو **SSH**. ستصبح هذه الاتصالات متاحة فقط بعد تثبيت نظام التشغيل وتكوينه بنجاح.

:::
