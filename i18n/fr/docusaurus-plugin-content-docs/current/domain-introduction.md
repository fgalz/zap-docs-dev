## Introduction

Vous cherchez un domaine à un prix équitable ? Alors ZAP-Hosting est l'endroit qu'il vous faut ! Chez ZAP-Hosting, vous pouvez enregistrer votre domaine pour individualiser votre présence en ligne. Vous pouvez par exemple utiliser le domaine pour un site web ou comme redirection pour un serveur de jeu ou un service de serveur vocal. 

![](https://screensaver01.zap-hosting.com/index.php/s/ESkLyeoLMTTENHG/preview)

## Commande

Le processus de commande est très simple. Choisissez votre configuration souhaitée pour votre [Domaine](https://zap-hosting.com/en/shop/product/domain/). Le temps de configuration ne prend que quelques secondes. Ensuite, vous pouvez immédiatement commencer la configuration. Plus d'informations sur le processus de commande exact se trouvent dans notre guide [Commander un Domaine](domain-order.md).
